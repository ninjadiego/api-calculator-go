# api-calculator-go

API REST sencilla en Go que expone operaciones matematicas basicas como suma, resta, multiplicacion y division. Proyecto de aprendizaje para practicar la construccion de endpoints HTTP en Go.

## Caracteristicas

- Suma de dos numeros
- Resta de dos numeros
- Multiplicacion de dos numeros
- Division de dos numeros (con validacion de division por cero)

## Tecnologias

- Go (Golang)
- HTTP estandar de Go (net/http)

## Como ejecutarlo

git clone https://github.com/ninjadiego/api-calculator-go.git
cd api-calculator-go
go run main.go

La API quedara corriendo localmente y podras hacer peticiones a los endpoints de cada operacion.
